quick_read(io_or_path, mode='r')

Read the contents of the passed object, opening and closing if required.

@param [read, String] a readable object or a filepath (String) @param [String] the reading mode, by default ‘r‘

This is a convenience function, allowing simple one-liners that read the contents of a file or similar object. If a string is passed, it is assumed to be a filepath, which is opened, read and closed. All other objects are presumed to be "readable"

@example 

   data = quick_read('foo.txt')

   hndl = File.open('bar.txt', 'rb')
   data = File.open(hndl)
   hndl.close()
