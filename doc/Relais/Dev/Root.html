- (Object) defaults(*args)

Create an options object with these default values.

Examples:

  def myfunc (arg1, arg2, opts={})
     options = defaults(
       :overwrite_data => true,
       :message =>  "foo",
     ).update!(opts)

See Also:

  • This is a simple bit of semantic sugar for defining the default values for methods. It just creates and returns an Options object with the passed values.
