- (Object) quick_read(io_or_path, mode = 'r')

Read the contents of the passed object, opening and closing if required.

Examples:

   data = quick_read('foo.txt')

   hndl = File.open('bar.txt', 'rb')
   data = File.open(hndl)
   hndl.close()

Parameters:

  • (#read, String) a — readable object or a filepath (String)
  • (String) the — reading mode, by default ‘r’ This is a convenience function, allowing simple one-liners that read the contents of a file or similar object. If a string is passed, it is assumed to be a filepath, which is opened, read and closed. All other objects are presumed to be "readable"

def quick_read (io_or_path, mode='r')
  rdr = BaseReader.new(io_or_path, mode)
  data = rdr.hndl.read()
  rdr.close()
  return data
end

def quick_write (data, io_or_path, mode='w')
  wrtr = BaseWriter.new(io_or_path, mode)
  wrtr.hndl.write(data)
  wrtr.close()
  return nil
end
